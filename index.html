Portfolio

Cook me though

The project is a responsive app for those who search food recipes on the Internet. It is based on external api end MVC model when it comes to client's side. If a user find some dish that wants to cook, can add ingredients to the shop list.

War of dices

It is a simple dice game for two. Players alternately keep rolling the dice. The one to get greater number wins and takes points of the opponent. A 3d animation of the dice was created to imitate a real situation. The game is finished after both players use 10 dices.

Typing Contest

The concept of a typing game was changed into an application that allow their users to solidify English vocabulary during the process of learning. A user have the opportunity to provide and customize own words list. The whole service is based on CRUD functionality and REST api.

Words Beast

The project was made after I read a book about the art of fast reading: Tom Buzan's "Speed Reading". A "pointer method" method was chosen and is the base of the app. To proceed with an animation I used canvas to create an indicator in the form of a red rectangle that slides among a text.
